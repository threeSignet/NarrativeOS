## Part 0: 设计系统基础

### 0.1 设计原则

#### 0.1.1 核心理念

NarrativeOS v3.0 Sovereign 的UI设计遵循以下五条核心原则：

| 原则 | 英文代号 | 说明 |
|------|----------|------|
| **叙事优先** | Narrative-First | 阅读与创作区域永远占据视觉重心，UI元素绝不喧宾夺主 |
| **空间即秩序** | Space is Order | 大量留白不是浪费，而是信息层级的视觉表达 |
| **暗黑即温柔** | Dark is Gentle | 深色模式不是反转颜色，而是为长时间创作提供护眼光照 |
| **反馈即对话** | Feedback is Dialogue | 每个操作都有明确反馈，系统与用户之间形成对话感 |
| **克制即力量** | Restraint is Power | 减少视觉噪音，每个元素的存在都有明确理由 |

#### 0.1.2 三模式哲学

系统围绕三种工作模式构建，每种模式有独立的视觉氛围：

| 模式 | 视觉隐喻 | 氛围描述 | 核心配色 |
|------|----------|----------|----------|
| **驾驶舱 Cockpit** | 星舰舰桥 | 专注、沉浸、精密 | 深空蓝灰 + 琥珀金 |
| **仪表盘 Dashboard** | 全息作战室 | 宏观、掌控、信息流 | 深灰 + 多点霓虹 |
| **休眠舱 Hibernation** | 静谧深空 | 极简、等待、呼吸 | 近黑 + 微弱脉动 |

#### 0.1.3 视觉层级

```
Level 1 — 内容层（正文/创作区）：最高优先级，最大面积
Level 2 — 工具层（操作栏/工具面板）：悬浮，半透明背景
Level 3 — 导航层（侧边栏/顶部栏）：缩略，常态可见
Level 4 — 系统层（通知/状态指示）：最小面积，瞬时出现
```

### 0.3 字体规范

--font-sans-zh: "Noto Sans SC", "Source Han Sans SC", "PingFang SC", "Microsoft YaHei", "Hiragino Sans GB", sans-serif;
